The Box Model Interactive Lab

Box Model Components (from inside out):
1. Content: The actual content (text, images, etc.)
2. Padding: Space between content and border (transparent)
3. Border: The edge surrounding the padding
4. Margin: Space outside the border (transparent, affects spacing between elements)

This box has all four layers!

Total Width: 250px

border-box (recommended)

Total width: 200px

Why border-box is better: When you set width to 200px, it stays 200px regardless of padding/border. This makes layouts predictable and easier to manage.

Pro Tip:Use * { box-sizing: border-box; } at the start of your CSS!

Example 5: Shorthand Notation

Centered with auto margins
Shorthand patterns:
  • padding: 10px; → all sides
  • padding: 10px 20px; → vertical horizontal
  • padding: 10px 20px 30px 40px; → top right bottom left (clockwise)
  • margin: 0 auto; → horizontally center block elements
